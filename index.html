About Wolf Pack

Wolf Pack is a War Thunder assistant, built for those that play together in a squad.

This works currently only for Aerial Battles, using the War Thunder provided APIs.

If you have any problems please raise an issue on the github project page which can be found at https://github.com/paulkilla/WolfPack
